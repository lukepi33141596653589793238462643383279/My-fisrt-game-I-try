# Pacman Game

A classic arcade-style game inspired by Pac-Man.

## Project Objective

This project was created for:
- game development practice
- software engineering studies
- logic and algorithms training
- 2D rendering experimentation
- artificial intelligence basics

## Technologies

- HTML5
- CSS3
- JavaScript
- HTML5 Canvas API

## Current Features

- Game window
- Rendering system
- Main game loop

## Planned Features

- Player movement
- Maze system
- Pellets
- Ghost AI
- Collision system
- Score system
- Lives system
- Game Over screen
- Sound effects
- Multiple levels
